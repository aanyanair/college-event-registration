# College Event Registration System 🎉

A web-based event registration system built using:

- Python Flask
- MySQL (XAMPP)
- HTML, CSS, JavaScript

## Features

- Register students
- Prevent duplicate KU IDs
- Store data in MySQL
- Clean UI

## How to Run

1. Install dependencies:
pip install flask mysql-connector-python

2. Run:
python app.py

3. Open:
http://127.0.0.1:5000
